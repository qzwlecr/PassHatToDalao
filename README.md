# PassHatToDalao

一个简单的向大佬递帽的工具

## Function

输入一个头像，输出给头像上戴上*圣诞帽子*的图片。

## Dependencies

  - OpenCV & CImg 图像处理库
  
## Usage

  - ./main 头像名称 [-o 输出名称]
  - config.cfg 具有可以调整的参数
## Notice

CImg已经附带。使用者需要手动安装OpenCV，否则无法通过编译。

## Build Methods

  - make
